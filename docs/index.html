UI Flow

點下 tab 之後發生的事

每次切 zone 都是一連串連動的事件 — 動畫、AI、UI 同步切換

1
👆

User 點 Tab

ZoneTabBar 接 onClick → 通知 ZoneManager

2
📷

Camera 跟拍

CameraRig.StartFollow → 沿路徑方向計算 offset → 從後方跟拍

3
🚶‍♀️

Navigator 移動

轉向 → 播 Walk → 直線走 → 隨機刷寶箱 → 抵達

4
🎬

Camera 定位

StopFollow → 平滑 lerp 到 zone 固定機位（unity-chan + 家具入鏡）

5
🧠

AI + UI 換

System prompt 換、對話清空、UI panel 切到對應 mode

CODE FLOW (簡化)
tabBar.onClick(zone)
  → zoneManager.RequestZone(zone)
      → cameraRig.StartFollow(unityChan, pathStart, pathEnd)
      → navigator.MoveTo(zone.anchorPos, zone.anchorYaw)
            ↳ OnPathStarted → treasureSpawner.Spawn?
            ↳ 每幀: transform.position += moveSpeed * dt
            ↳ OnPathArrived
      → cameraRig.SwitchTo(zone.cameraPos, ..., 1.2s)
      → aiManager.ApplyZone(zone)
            ↳ llm.SetSystemPrompt(zone.prompt, clearHistory=true)
      → OnZoneActivated
            ↳ panelSwitcher: enable target.uiMode panel
            ↳ tabBar: highlight active tab
